Word definitions for everyone

Here is a compressed directory of 340653 files. Each file is a word and contains corresponding definitions in it.

use:

If you have dmenu installed, make a script to open a file from .words directory with preferred pager:
#!/bin/sh
word=$(echo "" | dmenu -p "what word?")
$TERMINAL -e less ~/.words/"$word"
save it as %myfile% and run chmod 755 %myfile%
